Getting Started

### Install package: 
```
yarn install
```

<!-- dev -->

### Run the development server: 
```
yarn dev
```

<!-- stag -->

### Run the staging server: 
```
yarn stag
```

<!-- prod -->

### Run the prod server:
```
yarn prod
```

<!-- codegen -->

### Auto generate code for testing and change url location in `package.json`
```
yarn codegen
```

<!-- Report -->
### Report

yarn report
